# Study Planner

A feature-rich Study Planner built with Vanilla JavaScript that helps students organize, track, and manage their study tasks efficiently.

## Features

### Task Management

* Add new study tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed or pending

### Task Details

* Title
* Description
* Category
* Priority Level
* Due Date

### Search & Filtering

* Live search by task title
* Live search by task description
* Filter tasks by:

  * All Tasks
  * Completed Tasks
  * Pending Tasks

### Statistics Dashboard

* Total Tasks Count
* Completed Tasks Count
* Pending Tasks Count
* Clickable statistics for quick filtering

### Data Persistence

* Tasks are stored in Local Storage
* Data remains available after page refresh
* Automatic save on every state change
